# SigProfilerSingleSamplePro
New Version of SigProfilerSingleSample. SigProfilerSingleSample allows attributing a known set of mutational signatures to an individual sample. The tool identifies the activity of each signature in the sample and assigns the probability for each signature to cause a specific mutation type in the sample. The tool makes use of SigProfilerMatrixGenerator, SigProfilerExtractor and SigProfilerPlotting.
## Installs
for installing dependencies in new conda environment
```
pip install -r requirements.txt
```

Installing this package : git clone this repo or download the zip file.
Unzip the contents of SigProfilerExtractor-master.zip or the zip file of a corresponding branch.

```bash
$ cd SigProfilerSingleSamplePro-master
$ pip install .
```
